# Tourist Destinations and Hidden Gems

## Overview
Discover and explore popular tourist destinations and lesser-known hidden gems with this interactive platform. This project provides detailed information, images, and real-time maps for various places while allowing users to share feedback, like destinations, and plan trips collaboratively.

## Key Features
- **Explore Places**: View detailed information, descriptions, and images for destinations.
- **Interactive Maps**: Get real-time directions for locations.
- **User Feedback**: Leave comments and rate destinations with a 5-star system.
- **Favorite Destinations**: Like and save your favorite spots.
- **Trip Planning**: Plan trips or join existing ones, complete with group chat rooms for communication.

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Flask
- **Database**: SQLAlchemy with SQL
- **Other Tools**: CSS , React for styling, Google Maps API for real-time directions

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/username/tourist-destinations.git
   ```
2. Navigate to the project directory:
   ```bash
   cd tourist-destinations
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the Flask app:
   ```bash
   flask run
   ```

## Usage
- Access the platform at `http://127.0.0.1:5000` after starting the Flask server.
- Browse destinations, rate places, and share feedback.
- Plan trips and communicate with fellow travelers.
